This is the landing page for HowdyBuzz, built with HTML and Tailwind CSS.

## Setup
No installation is required. Simply open `index.html` in your web browser.

## Features
- **Hero Section**: Features a fallback image (`assets/student_hero.png`).
- **Daily Mastery Card**: A glassmorphism feature card with an ID of `daily-content-hook` for your Google Apps Script integration.
- **Exam Grid**: A responsive grid for GRE, GMAT, IELTS, SAT, and TOEFL tracks.

## customization
- **Tailwind CSS**: The styles are loaded via CDN for immediate usage.
- **Colors**: Custom colors (Midnight Blue: `#0b1a2a`, Brand Gold: `#FFD700`) are defined in the `tailwind.config` script within `index.html`.

## Images
- `assets/student_hero.png` is included as the video poster/fallback.
